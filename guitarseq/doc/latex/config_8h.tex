\subsection{Detailed Description}
Configuration file processing library. This library is meant to be short and simple for parsing ini style files. The files should consist of \char`\"{}key = value\char`\"{} pairs, though \char`\"{}key value\char`\"{} and \char`\"{}key : value\char`\"{} pairs, as long as the key has a whitespace after Lines starting with '\#' are ignored
